# How to use

<h2>Step 1: download in your system.</h2>  

git clone https://github.com/siddharth018/laravel8autocomplete.git

    cd laravel8autocomplete

<h2>Step 2: Configure your database from .env file</h2> 

   
    DB_HOST=127.0.0.1

    DB_PORT=3306
    
    DB_DATABASE=laravel8autocomplete
    
    DB_USERNAME=root
    
    DB_PASSWORD=root@123

<h2>Step 3: composer install</h2> 
    
    composer install

<h2>Step 4: Migrate</h2> 
    
    php artisan migrate

<h2>Step 5: key generate</h2> 

    php artisan key:generate

<h2>Step 6: Run Seeder</h2> 

    php artisan seed:db

<h2>Step 7: Run server</h2>  

    php artisan serve

<h2>Step 8: Search Now</h2>  

    http://127.0.0.1:8000/search
